SUD data

In version 2.9 of SUD data, released in December 2021:

  • 5 corpora are built in the SUD format (called Native SUD)
  • 212 corpora of UD 2.9 are automatically converted to SUD from the corresponding UD data.

In the table below, the 5 native SUD corpora are given. Note that each corresponding UD version is obtained by automatic conversion.

SUD data for version 2.9

All UD (version 2.9), except the 5 ones coming from native SUD (UD_Beja-NSC, UD_French-GSD, UD_French-Rhapsodie, UD_French-ParisStories and UD_Naija-NSC) are converted in SUD format with the UD â†’ SUD conversion system (tag 2.9) (usable with the Grew software).

Hence, the data below are built with:

  • The 5 native SUD described above
  • 212 corpora converted:
    • from version 2.9 of UD corpora
    • with tag v2.9 of the conversion system
