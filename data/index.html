In version 2.7 of (S)UD data, released in November 2020:

  • 3 corpora are maintained in the SUD format (called Native SUD)
  • All 183 corpora of UD 2.7 are automatically converted to SUD from the corresponding UD data.

In the table below, the 3 native SUD corpora are given. Note that each corresponding UD version is obtained by automatic conversion.

NB In order to give a consistent set of corpora, the 3 native SUD corpora also appear in this list. But the data below for these 3 corpora is the result of the conversion from the UD version (hence a double conversion SUD ‚Üí UD ‚Üí SUD). The concerned corpora are marked with a “‚ö†Ô∏è” symbol.

The 183 UD (version 2.7) corpora are converted in SUD format with the UD ‚Üí SUD conversion system (usable with the Grew software).

The data below are built with:

  • version 2.7 of UD corpora
  • tag v2.7 of the conversion system
